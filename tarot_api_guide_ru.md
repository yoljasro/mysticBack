# Tarot API Guide (Карта Таро на сегодня)

Этот документ описывает API для раздела "Фортуна / Карта Таро на сегодня".

Базовый URL: `/api/tarot`

Все запросы (кроме авторизации) требуют заголовок `Authorization: Bearer <token>`.

---

## 1. Получить карту Таро (Draw Daily Card)
Этот эндпоинт используется, когда пользователь заходит на экран "Карта Таро на сегодня" и открывает карту. Возвращается случайная карта Таро со значением и любовным прогнозом.

**URL**: `/api/tarot/daily`

## 2. Вытянуть еще раз (Draw Again)
Этот эндпоинт используется при нажатии на кнопку "Вытянуть еще раз". Функционально он работает так же, как и получение ежедневной карты, но отделен на случай, если в будущем потребуется отслеживать количество попыток или взимать за это монеты/оплату.

**URL**: `/api/tarot/draw`  
**Метод**: `POST`  
**Заголовок**: `Authorization: Bearer <token>`

### Ожидаемый ответ (200 OK):
```json
{
  "success": true,
  "data": {
    "card": {
      "_id": "64abcdef1234567890abcd34",

        "strongPoint": "Связь",
```

## Возможные ошибки

### Ошибка 404 (Карты не найдены)
Если база данных карт пуста.
```json
{
  "success": false,
  "message": "No tarot cards found in the database."
}
```

### Ошибка 401 (Не авторизован)
Если токен не передан или недействителен.
```json
{
  "success": false,
  "message": "Not authorized to access this route"
}
```

### Ошибка 500 (Ошибка сервера)
```json
{
  "success": false,
  "message": "Server Error"
}
```

## 3. Список всех карт (Get All Cards)
Возвращает список всех доступных карт в базе данных.

**URL**: `/api/tarot/cards`  
**Метод**: `GET`  
**Заголовок**: `Authorization: Bearer <token>`

### Ожидаемый ответ (200 OK):
```json
{
  "success": true,
  "count": 5,
  "data": [
    { "_id": "...", "name": "Карта Звезды", ... },
    { "_id": "...", "name": "Влюбленные", ... }
  ]
}
```

---

## 4. Детали конкретной карты (Get Card by ID)
Возвращает информацию о конкретной карте по её ID.

**URL**: `/api/tarot/cards/:id`  
**Метод**: `GET`  
**Заголовок**: `Authorization: Bearer <token>`

### Ожидаемый ответ (200 OK):
```json
{
  "success": true,
  "data": {
    "_id": "64abcdef1234567890abcd12",
    "name": "Карта Звезды",
    ...
  }
}
```

---

## 5. Расклад из трех карт (Three Card Spread)
Выбирает 3 случайные уникальные карты для расклада "Прошлое, Настоящее, Будущее".

**URL**: `/api/tarot/spread/three`  
**Метод**: `GET`  
**Заголовок**: `Authorization: Bearer <token>`

### Ожидаемый ответ (200 OK):
```json
{
  "success": true,
  "data": {
    "spread": [
      { "position": "Past", "card": { ... } },
      { "position": "Present", "card": { ... } },
      { "position": "Future", "card": { ... } }
    ]
  }
}
```
